# Zymkey NodeJS Library

A NodeJS wrapper for zymkey crypto library.

## Installation

```bash
npm install zymkey --save
```

## Usage 

```nodejs
var client = require('zymkey');
var ret = client.ledOff();
console.log('Zymkey ledOff returned ' + ret);
```
